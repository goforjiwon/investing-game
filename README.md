# 5인 공동투자 게임

투자와 사회 구조를 체험하는 시뮬레이션 사회 실험 게임입니다.

## 게임 개요
- 5명의 고정 플레이어(P1~P5)가 10라운드 동안 반복 플레이
- 라운드마다 각자 10토큰 지급
- 각자 0~10토큰 정수로 비공개 투자
- 공동기금 총액에 2배 배수를 적용한 뒤 5명에게 균등 분배
- 개인 점수와 누적 점수를 통해 협력/무임승차 행동 관찰

## 라운드 점수 공식
`라운드 점수 = 10 - 내가 낸 토큰 + (전체 투자 합계 × 2 ÷ 5)`

## 실행 방법
브라우저에서 `index.html` 파일을 열면 바로 플레이할 수 있습니다.

## 포함된 기능
- 익명 코드 기반 표시(P1~P5)
- 라운드 제한 시간 설정
- 라운드 결과 공개:
  - 내 투자액
  - 전체 투자 총액
  - 2배 적용 후 총 분배액
  - 내가 받은 분배액
  - 내 라운드 점수 / 누적 점수
- 권장 공개 정보:
  - 플레이어별 익명 투자액
  - 그룹 평균 투자액
  - 현재 라운드 번호
- 최종 결과판(플레이어별 누적 점수, 평균 투자액)
