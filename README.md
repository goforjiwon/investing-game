# 5인 협력 심리전 투자 게임

공공재 게임(사회실험형)의 핵심 구조를 유지하면서, 친구들과 온라인에서 심리전 중심으로 즐길 수 있게 게임화한 버전입니다.

## 고정 규칙
- 5인 고정 멀티플레이
- 친구 초대 코드 기반 방 생성
- 총 7라운드
- 라운드당 20초 제한 시간
- 매 라운드 10토큰 지급
- 0~10토큰 비공개 투자(미입력 시 자동 0 처리)
- 전체 투자금 2.5배 증폭 후 5명 균등 분배
- 닉네임별 투자액 공개
- 누적 점수 리더보드 표시

## 추가 게임화 요소
- 제한형 메시지 기능
- 라운드 시작 전 목표 기여량 선언
- 전원 5토큰 이상 투자 시 협력 보너스 전원 +2점
- 라운드 종료 후 각 플레이어는 1명에게 경고표 1개 부여
- 경고표 2개 이상 누적 플레이어는 다음 라운드 시작 토큰 -2 페널티

## 결과 화면 정보
- 내 투자액 / 내 라운드 점수
- 다른 플레이어 투자액(닉네임 공개)
- 전체 투자 총액
- 배수 적용 후 총 분배액
- 누적 점수 리더보드
- 최종 결과: 우승자, 협력왕, 전체 평균 기여율

## 실행 방법
브라우저에서 `index.html`을 열면 바로 실행됩니다.
